HTML Cheat Sheet

Tag Usage 
<!DOCTYPE html>
 Declares the document as a HTML file 
<HTML>
 All code must be within this element 
<head>
 Contains the metadata e.g title, links 
<body>
 Contains code that can be viewed on the webpage 
<header>
 Used for the header section 
<nav>
 Used for the navigation bar 
<h1>
 Used when creating a heading. h1-h6 
<a>
 This element stands for anchor tag. It is used to link to other pages or sections 
<img>
 This is the image tag and is used to input images into your code 
<main>
 Semantic element that allows you to identify the main section of your page 
<footer>
 The bottom section of the page 
<article>
 Contains blogs, articles, comments, etc. 
<section>
 Defines a new element on the page 
<aside>
 Used for non important information 
<figure>
 Used when referencing another section. e.g an image of someone mentioned in the main text 
<figcaption>
 Caption for the figure element 
<audio>
 Used to provide audio on the page 
<video>
 Used to provide a video on the page
